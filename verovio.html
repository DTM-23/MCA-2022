One element which can be compared is how XML simply states the duration of a note with a single number. On the other hand, MEI uses PPQ or notes per quaver in order to measure durations. A duration of “4” is equal to that of a minum, as it has 4 quavers within it.

A second element to compare is the use of width and coordinates within the XML file. This file uses width to individually size each bar, and uses x and y to place notes. This is not present on the MEI file, which does this seemingly automatically.

A third element to compare is the length of each individual file. The MEI file is much more organised and condensed than the XML file, taking up 1224 lines of code compared to the massive 5431 of the XML file.
